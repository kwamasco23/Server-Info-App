# Server Info App (DevOps Training Project)

Interactive Python Flask application designed to simulate real-world DevOps workflows.

This project is used to practice:

- Docker containerisation
- GitHub version control
- GitHub Actions CI/CD
- Jenkins pipelines
- Docker Hub registry
- EC2 deployment
- On-prem deployment
- Logging and monitoring

---

# Application Features

- Interactive web form
- Displays:

  - Version
  - Environment
  - Hostname
  - IP Address
  - Build number
  - Platform
  - Timestamp

- Health endpoint
- API endpoint
- Request logging (GET and POST)
